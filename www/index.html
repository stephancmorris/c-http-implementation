Welcome to NanoServe - a lightweight, multi-threaded HTTP server written entirely in C from scratch!

Features

  • Thread Pool Architecture
  • HTTP/1.1 Protocol Support
  • Static File Serving
  • MIME Type Detection
  • Concurrent Request Handling
  • Thread-Safe Logging

Architecture

NanoServe uses a producer-consumer pattern with:

  • Listener: Binds to port 8080
  • Dispatcher: Accepts connections
  • Queue: Thread-safe FIFO task queue
  • Workers: Pool of threads processing requests
